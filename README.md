# stellar-app-eventticket
# 🎟️ Event Ticketing System (Soroban on Stellar)

## 📌 Project Description
The Event Ticketing System is a decentralized application (dApp) built on the Stellar blockchain using Soroban smart contracts.  
It allows users to create events, buy tickets, and transfer ticket ownership in a secure and transparent way.

This project demonstrates how blockchain technology can be used for managing event tickets without relying on centralized platforms.

---

## ⚡ What This Project Does
This system enables:

- Event organizers to create events on-chain
- Users to browse available events
- Purchase tickets securely
- Transfer tickets to other users
- Track ticket ownership on the blockchain

All transactions are processed on the Stellar Testnet using Soroban smart contracts.

---

## ✨ Features
- Create events
- View all available events
- Buy tickets
- Transfer tickets
- Decentralized ownership
- Low transaction cost
- Fast finality (~5 seconds)
